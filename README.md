# Theo Ross — Portfolio Website

A portfolio site built with HTML, CSS, and vanilla JavaScript. Designed for hosting on **GitHub Pages**.

## Project Structure

```
theo-ross-portfolio/
├── index.html          # Home
├── resume.html         # Resume
├── projects.html       # Projects
├── experiences.html    # Experiences & Interests
├── contact.html        # Contact form
├── css/
│   └── style.css       # All styles
├── js/
│   └── main.js         # Nav, scroll reveal, contact form
└── assets/             # PDFs and Images
```

## Deploying to GitHub Pages

1. Create a new GitHub repository (e.g. `theo-ross.github.io` or any name)
2. Upload all files keeping the folder structure intact
3. Go to **Settings → Pages**
4. Set Source to **Deploy from a branch**, select `main`, folder `/root`
5. Click Save — your site will be live at `https://<username>.github.io/<repo>/`
